Succulent Match, is a memory game I have made for my course project for Code Institue in order to demonstrate my javascript knowledge.

I chose succulents as the subject matter for my project as I am passionate about these plants due to the many different varieties you can find, as you will see through playing the game. Succulents vary in shape, size, colour and for this reason I find them very interesting and have started my own collection.

Hopefully this game will spark an interest in these strange but interesting plants for others as well!

How to play...

The object of the game is to find the hidden pairs in the quickest time and in the smallest number of moves

You will get 2 clicks per go to reveal 2 cards of your choice to see if they match

This game is all about improving memory so dont worry if your pairs dont match at first, just remember where they are in the grid for future turns!

There are 3 levels of difficulty for the game:

  • Easy = 6 pairs to find
  • Medium = 8 pairs to find
  • Hard = 10 pairs to find

Good luck!
